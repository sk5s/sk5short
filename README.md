## Getting Started

### Development

#### clone
```bash
git clone https://github.com/sk5s/sk5short.git
cd sk5short
```

#### vite
[https://vitejs.dev/](https://vitejs.dev/)

```bash
npm install
npm run dev
```

#### build page
```bash
npm run build
```
